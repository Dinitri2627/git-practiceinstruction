## Глава 1: Введение и Начальная Настройка

Создание аккаунта на GitHub и установка Git на ваш компьютер.

### 1.1. Регистрация на GitHub

GitHub — это веб-сервис для хостинга IT-проектов и их совместной разработки. Основан на системе контроля версий Git.

1.  Перейдите на сайт [github.com](https://github.com/).
2.  Нажмите кнопку **"Sign up"**.
3.  Следуйте инструкциям на экране: введите вашу почту, придумайте пароль и имя пользователя.
4.  Подтвердите свой аккаунт через письмо, которое придет на указанную почту.

### 1.2. Установка Git

Git — это программа, которая устанавливается на ваш компьютер и выполняет контроль версий.

1.  Перейдите на официальный сайт [git-scm.com](https://git-scm.com/).
2.  Скачайте установочный файл для вашей операционной системы (Windows, macOS, Linux).
3.  Запустите установщик. Для большинства пользователей подойдут настройки по умолчанию.
4.  После установки откройте терминал и проверьте, что Git установлен корректно.

**Пример проверки версии:**
\`\`\`bash
$ git version
git version 2.51.0.windows.1
\`\`\`
**Необходимо:** вам нужно ввести свои данные в Git, чтобы коммиты подписывались вашим именем.
\`\`\`bash
$ git config --global user.name "Имя"
$ git config --global user.email "Почта@example.com"
\`\`\`
